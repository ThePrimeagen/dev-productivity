### BUT DID YOU KNOW??
```
tmux new-session -s "foobar" -d -c "$HOME/personal/developer-productivity"
```
```
tmux new-window -n "foobar" -c "$HOME/personal"
```

### We can specify a command.
```
tmux new-window -n "foobar" [some command goes here]
```
